# Сервис для работы с БД

## Файловая архитектура
```
bd_service
├── tables/                # Папка, где хранятся md файлы с информацией про таблицы, название файла - название таблицы
└── Data.md                # Файл с кратким описанием какие данные нужны банку
```

## Выбор БД
 Для хранения полей и пользователской информации используется Postgre. В Postgre будут следующие БД: 
  1. Учётные данные пользователя
  2. История операций пользователя 
 Для хранения логов используется ClickHouse.
 Для хранения токенов используется Redis.
 Для хранения уведомлений MongoDB.
