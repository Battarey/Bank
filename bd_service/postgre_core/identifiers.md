# identifiers

Содержит ИНН и СНИЛС, подтверждающие личность клиента. Таблица ссылается на `users` и используется сервисами KYC.

| Column    | Type    | Constraints                      | Description                              |
|-----------|---------|----------------------------------|------------------------------------------|
| client_id | UUID    | PK, FK -> users.id, NOT NULL     | Связан с записью клиента.                |
| inn       | CHAR(12)| UNIQUE, NOT NULL                 | ИНН, хранится без пробелов.              |
| snils     | CHAR(11)| UNIQUE, NOT NULL                 | СНИЛС, очищенный от разделителей.        |
