# Сервис для работы с аккаунтом/счетами пользователя

 Банковский аккаунт даёт доступ пользователю к созданию счёта в банке.
 Счёт непосредственно создан для манипуляций с деньгами.

## Файловая архитектура
```
bank_account_service
├── bank_account/                # Работа с счётом пользователя
├── user_account/                # Работа с банковским аккаунтом
├── Dockerfile
├── README.md
└── requirements.txt
```
